<h2>ðŸ“Œ Overview</h2>

MedSync is a production-grade web application designed to manage doctor appointments efficiently.  
It offers patient onboarding, doctor schedules, appointment booking, email notifications, document uploads, and secure authentication.

The system uses:
- **Angular** for a modular Single Page Application (SPA)
- **ASP.NET Web API** for backend business logic
- **SQL Server** for relational data
- **JWT Authentication** for security
- **Class Libraries** for DAL, Models, and Helpers
